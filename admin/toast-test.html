Toast Webhook Test
Send a fake Toast order to your live toast-order-webhook function. Use this from anywhere to verify the integration without ringing up a real transaction at the bar.

2. Test Order
Pretend a member just closed a tab. The webhook will look up the member by email (or phone if no email) and credit points based on the amount × earn rate × tier multiplier from rewards.json.
Must match an existing member in members.json. Add yourself first via Admin → Customers → Rewards Members.
Tips
• Reuse the same Order ID twice — second attempt should be skipped (idempotency check working).
• Try an email that's NOT a member — should return unmatched (failsafe working).
• Try $1, $50, $300 — verify points scale correctly (10 pts per $1 default).
• Cross 3,000 lifetime points to verify tier promotion to Gold; 6,000 → Platinum.
• Live logs: Netlify → Functions → toast-order-webhook → Logs.
